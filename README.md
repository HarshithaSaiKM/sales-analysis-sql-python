# Sales Data Analysis using SQL and Python

## Project Overview
This project analyzes sales data using SQL and Python to find:
- Total sales by category
- Monthly sales trends
- Top-performing products

## Tools Used
- Python (Pandas, Matplotlib)
- SQL
- Jupyter Notebook

## Steps Performed
1. Loaded dataset
2. Performed SQL queries
3. Visualized results using Python

## Output
Bar charts and insights from sales data.
